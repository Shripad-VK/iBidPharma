Run Angular + Spring Boot Project 
1. For execution policy error command : Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope CurrentUser
2. For running imported angular project : npm install --save-dev @angular-devkit/build-angular
3. Running angular Project : ng serve --proxy-config proxy.config.json
4. For running our project : npm start
5. command for bootstrap installation in angular  :  npm install bootstrap jquery popper .js --save
